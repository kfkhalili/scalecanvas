# Google Cloud icons

SVGs in this folder are from the official **Google Cloud Icon library** (2025 set):

- **Source:** [Google Cloud Icon library](https://cloud.google.com/icons)
- **Sets used:** Core product icons + Product category icons (updated early 2025). Legacy console icons are not used.
- **Purpose:** Hosted for use in app architecture diagrams.

Use of these icons is subject to [Google Cloud’s brand and asset guidelines](https://cloud.google.com/terms).
